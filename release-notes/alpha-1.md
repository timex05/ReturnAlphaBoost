# ReturnAlphaBoost Alpha 1

This first alpha release includes:

- Auto-detected Rocket League install path
- Manual install path selection
- Copy of alpha files from the repository's `alpha_files` folder into `TAGame\CookedPCConsole`
- Overwrite support for existing files
- Close-Rocket-League startup warning
- Simple UI focused on the copy workflow

This is an alpha build and may change frequently.
